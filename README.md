# TxtToXml

Uygulamanın amacı: Uygulama çalıştığı sürece belirli zaman aralıkları ile işletim sistemi üzerinde bir dizini dinleyerek, bu dizine kopyalanan txt dosyasını işleyerek, dosya içerisindeki bir sınıflandırma bilgisine göre çıktı olarak xml dosyaları oluşturacak.
 
Senaryo:
 
1.       Input-data.txt(Farklı isimde olabilir) dosyası dinlenen dizine (Örn: C:\listen) kullanıcı tarafından kopyalanır.

2.       Uygulama çalıştığı sürece belirli zaman aralıkları ile (Örn:30sn) dinlediği klasöre dosya bulduğunda dosyayı işleyecek prosesi başlatır.

3.       Input-data dosyası içerisindeki data “TXT5” değerine göre gruplanarak xml dosyaları işletim sisteminde bir dizinde(Örn: c:\output) oluşturulur.

4.       Uygulama işlemeyi bitirdiği input data dosyasını siler.
